# API Part

Method: POST
mimetype: json
Endpoint: /

## Request fields:

```
["short_description", "long_description", "lat", "long"]

```

### short_description:

a brief description of the problem (mandatory)

### long description:

A long desciption of the problem / issue. Optional

### lat:

GPS latitude coordinates

### long:

GPS longitude coordinates
